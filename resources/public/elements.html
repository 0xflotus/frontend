Text elements with images

To float an image to the right apply class="right" to the image. To float an image to the left apply class="left" to the image. If you wish for the image to have a border, apply class="inner_img". The hover effect over the image (doubling the border) is applied only when the image is inside an anchor tag.

So, the HTML code for the image in the following paragraph is <a href="#"><img class="inner_img right" src="..." width="..." height="..." alt="" /></a>

Happiness is a state of mind or feeling characterized by contentment, love, satisfaction, pleasure, or joy. A variety of biological, psychological, religious, and philosophical approaches have striven to define happiness and identify its sources. While direct measurement of happiness presents challenges, tools such as The Oxford Happiness Questione have been developed by researchers. Positive psychology researchers use theoretical models that include describing happiness as consisting of positive emotions and positive activities, or that describe three kinds of happines. Button elements

Buttons are styled by default using colors that match the current template you have selected. However, if you want a button to have a different look than the default, you can use the following classes: button_large_orange, button_large_green, button_large_yellow, button_large_brown, button_large_black. You also need to use the call to action class, so the code for a button could look like this: <a class="call_to_action button_large_orange" href="#">Button!</a>

* Note that if you want to use a different style than the default for your buttons, you have to add the "additional.css" file to the HTML page header like this: <link href="css/additional.css" rel="stylesheet" type="text/css" media="screen" />
